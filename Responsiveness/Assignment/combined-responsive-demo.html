Responsive Design

Responsive web design is an approach that makes web pages render well on various devices and screen sizes. It uses flexible layouts, images, and CSS media queries to adapt content to different viewing environments.

Quick Reference

  • Flexbox: display: flex;
  • Grid: display: grid;
  • Media Query: @media (max-width: 768px)
  • Clamp: font-size: clamp(1rem, 2vw, 2rem);
