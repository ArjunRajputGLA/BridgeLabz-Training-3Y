Responsive CSS Cheat Sheet

CSS Filters

Apply visual effects to elements like blur, grayscale, brightness, and more.

filter: blur(5px);

Media Queries

Apply styles based on device characteristics like screen width.

@media (max-width: 768px) { ... }

Responsive Design

Create layouts that adapt to different screen sizes using flexible units.

width: 100%; max-width: 1200px;

Typography

Use clamp() for fluid typography that scales smoothly.

font-size: clamp(1rem, 2vw, 2rem);

Flexbox

One-dimensional layout method for arranging items in rows or columns.

display: flex; gap: 20px;

CSS Grid

Two-dimensional layout system for creating complex responsive layouts.

display: grid; grid-template-columns: repeat(3, 1fr);
